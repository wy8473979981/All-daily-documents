# Title 标题组件

标题组件

## Title Attributes

| 参数    | 说明             | 类型    | 可选值 | 默认值 |
| ------- | ---------------- | ------- | ------ | ------ |
| title  |  标题文字  | String  | -      | -      | '' |

## Slot Attributes

`slot`为右侧自定义内容

## Demo

### 基本用法

```html
<template>
  <ysn-title title="基础数据">
     右边内容
   </ysn-title>
</template>
<script>
  export default {
    data() {
      return {
      }
    },
    methods: {
     
    }
  }
</script>
```
