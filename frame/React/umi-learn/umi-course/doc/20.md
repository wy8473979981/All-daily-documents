
dispatch可以把事件发布到reducers和effects，这里我们只需要更新filterKey就好，所以我们发起一个type为sava的事件，这是我们之前的代码了，可以再看一下，它只是把参数同步到state中
```javascript
reducers: {
    save(state, action) {
      return { ...state, ...action.payload };
    },
  },
```
